# Prioritized Quarterly Roadmap: Algeria Data Platform

This roadmap translates the strategic phases of the Algeria Data Platform project into a prioritized, actionable plan for the first year of development.

---

## **Q1 (Months 1-3): Foundation & MVP Launch**
**Objective**: Establish the core business and technical foundation, launch a Minimum Viable Product (MVP) to validate market demand, and secure the first cohort of users.

| Priority | Deliverable                               | Owner                    | Effort Estimate | Key Performance Indicators (KPIs)                                                                 |
| :------- | :---------------------------------------- | :----------------------- | :-------------- | :------------------------------------------------------------------------------------------------ |
| **1**      | **Legal Entity & Business Setup**           | Founder / CEO            | 1 dev-week      | - Legal entity (SARL/SPA) officially registered.<br>- Business bank account operational.                   |
| **2**      | **POC Implementation**                    | Lead Engineer            | 2 dev-weeks     | - Functional FastAPI application serving seed data.<br>- CI/CD pipeline established.                    |
| **3**      | **Hire Core Technical Team**              | CTO                      | -               | - Lead Engineer, Data Engineer, and Full-Stack Developer hired.                                   |
| **4**      | **Launch Public MVP**                     | Product Manager          | 6 dev-weeks     | - Platform live with 50+ datasets.<br>- User authentication & subscription billing functional.<br>- >500 registered users. |
| **5**      | **Secure Alpha & Beta Users**               | Business Development     | 2 dev-weeks     | - 20 alpha users providing initial feedback.<br>- 100 beta users onboarded for public testing.        |
| **6**      | **Establish Data Governance Committee**     | Chief Data Officer (CDO) | 1 dev-week      | - Initial members from public/private sectors recruited.<br>- First governance meeting held.     |
| **_Overall Q1 Goal_** | **Achieve Product-Market Fit Validation** | **Entire Team**            | **~12 dev-weeks** | **- 50+ paying customers.<br>- $10,000 Monthly Recurring Revenue (MRR).**                              |

---

## **Q2 (Months 4-6): Scalable Infrastructure & Data Expansion**
**Objective**: Evolve from an MVP to a scalable, robust data platform by implementing the Data Lakehouse architecture and significantly expanding the quantity and quality of data.

| Priority | Deliverable                                 | Owner                  | Effort Estimate | Key Performance Indicators (KPIs)                                                                    |
| :------- | :------------------------------------------ | :--------------------- | :-------------- | :--------------------------------------------------------------------------------------------------- |
| **1**      | **Data Lakehouse Implementation (v1)**      | Data Engineering Lead  | 8 dev-weeks     | - Bronze, Silver, Gold layers operational on hybrid cloud.<br>- 100+ datasets processed via automated pipelines. |
| **2**      | **Advanced Data Quality Framework**         | Data Engineering Lead  | 4 dev-weeks     | - 100+ Great Expectations tests running daily.<br>- Data quality score > 95% for all Gold datasets.     |
| **3**      | **Expand API & SDKs**                       | Lead Engineer          | 3 dev-weeks     | - GraphQL API launched.<br>- Python and JavaScript SDKs published.                                    |
| **4**      | **Real-Time Data Ingestion (Pilot)**        | Data Engineering Lead  | 3 dev-weeks     | - Apache Kafka cluster deployed.<br>- 5+ real-time data sources integrated (e.g., social media).      |
| **_Overall Q2 Goal_** | **Achieve Technical Scalability & Data Leadership** | **Engineering Team**   | **~18 dev-weeks** | **- 500+ datasets available.<br>- API p95 latency < 500ms.**                                            |

---

## **Q3 (Months 7-9): Intelligence Generation & Monetization**
**Objective**: Move up the value chain from providing data to delivering actionable intelligence. Deploy machine learning models and BI tools to create high-value, monetizable data products.

| Priority | Deliverable                                       | Owner               | Effort Estimate | Key Performance Indicators (KPIs)                                                                      |
| :------- | :------------------------------------------------ | :------------------ | :-------------- | :----------------------------------------------------------------------------------------------------- |
| **1**      | **Deploy Production ML Models**                   | Lead Data Scientist | 8 dev-weeks     | - LASSO-OLS GDP forecast model live.<br>- 5+ other predictive models deployed as API endpoints.          |
| **2**      | **Launch Public BI Dashboards**                   | Product Manager     | 4 dev-weeks     | - 20+ public dashboards (Metabase/Superset) embedded on the platform.<br>- 1,000+ active users.          |
| **3**      | **Multilingual NLP Pipeline**                     | Lead Data Scientist | 5 dev-weeks     | - Sentiment analysis for Algerian Arabic (Darja) achieving >90% accuracy.                             |
| **4**      | **Data Contributor Portal**                       | Lead Engineer       | 4 dev-weeks     | - Portal live, allowing users to submit data.<br>- First 10 contributor datasets published.           |
| **_Overall Q3 Goal_** | **Become an Indispensable Intelligence Tool**     | **Product & Data Team** | **~21 dev-weeks** | **- 5+ Enterprise clients.<br>- $50,000 Monthly Recurring Revenue (MRR).**                           |
